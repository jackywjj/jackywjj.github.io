角色的改变从名字开始

前几天逛facebook，突然看见前老板贴了张2005年的照片，看了一下他贴出来的说明，原来是tudou创业时的最初概况。

首先说明下，前老板应该是tudou的董事，投钱的金主，10年了，tudou成长得很快，前老板也发达了，租了个游轮到处玩，非常羡慕，很希望未来也有这样的生活。

看到这张照片，更坚定了我的信心，很多伟大的公司都来自于车库，前期艰苦的创业，并不是一句空话，唯有执着的坚持，偏执的前进，才能存活下来。 多年的技术积累和经历，身处各类公司、各类项目、各类团队中，多多少少看到过很多东西，也有许多思考，不同的时期，不同的人员情况、不同的技术情况，就如同大风大浪中前行的孤独的黑珍珠号，努力驰向彼岸，各种滋味，我想只有老板们才有感觉。

很长的技术生涯中，养成了一些特有的习惯，加上本身性格，特别喜欢新鲜事物，新鲜的说辞，新鲜的技能。这些习惯，帮助我能够和一些顶尖的工程师达成共识，能在一起和谐的工作，因此我觉得非常有用，当然也有很多人认为太过不正经，我也只能呵呵了。

不过，这并非没有道理，在启动我的新的职业生涯之际，是时候需要做一些转变，思考的范围、方式方法也需要更多改变，不能再局限在技术范畴了，更多的领域和范围需要考虑。特别要提醒自己，游走在各种需求之际，权衡做出决定的时候，尽可能不要短视，不能一步错，步步错。

坚持的动力

想了一下，从什么地方开始捏？首先，仿照前老板的做法，把这个moment 给记录下来，记住原始冲动，在很长的一段时间里，需要这股原始的动力不停的激励自己。

面子工程

第二，改变自己形象，这点从改昵称开始，小镇树妖， 这个昵称从大约2002年开始用起，后来就没有大改过，游戏中可能叫树妖法师，树灵法师之类，Blog使用树妖攻城湿之类，这些都是源于我对大自然，特别是原始森林的喜爱，以及对怪异的奇幻文学的钟意。再加上软件工程师的职业，就促成了这个昵称。

现在，是时候改变了，使用英文名 + 公司名 + 职位title，这种方式，感觉更好点，更正规，少一点搞笑的元素，多点认真的感觉，应该会加分不少。

如何运作

第三，思考的方式需要转变，不管是碰到产品问题，还是产品特性，不能马上从工程师角度去想，如何实现的？可能有什么原因？可能有什么方法去debug？可能有哪些解决方案等等。

替代这类思维的，可能需要更全局性的视角，更长远的考虑。全局的视角，需要权衡各个方面的因素，同时，加上长远的考虑，做出尽量最优的方案。实在是非常头痛的工作内~~~

总结一下，改变是必须的，要做的事情，其实应该也没有太大不同，我将之比喻成走钢丝，其实有很多人做这种比喻，实在是太贴切了。如何做法是个谜，很多时候自己都不知道，但是看人家做，又是很清晰的，果然是当局者谜。所以给自己一句话，偶尔停下来，看看自己走过的路。

以上写给未来的自己，孔子说过，改变从名字开始。

CJ阿鹤和我的故事

CJ 和 阿鹤都是靓仔，16岁青葱岁月的同窗。由于某些原因，做了些许回忆，每个人生命中都会有些难以忘怀的事物，这些事物往往牵涉到几个人，偶尔提起，大家都会非常高兴。

或许记录下来，未来单独翻看时，便不会有聚首时那般情景，我想以第三方的视角去描述，或许会更有趣。

分几个段落，平平淡淡的描述一下而已，留着未来细细回忆，特别怕家族中老年痴呆被遗传到，记录一下，虽然可能回忆的时候照样比较吃力，然而徜徉在如此温馨的场面中，我想未来的我也会不自禁的傻笑，为了提高逼格，略去猥琐部分不谈，只留真情所在。

先介绍下CJ，靓仔一枚，很爱漂亮，据说在卫生间可以打扮个把小时，洗刷、吹风、修面等等，嗯，是个非常注重仪表的小伙子，那么那个岁数，好像个把小时是必须的，青春少年，花个把小时洗个比较长时间的澡，也是很平常的。

阿鹤是个猥琐的主，可偏偏现在长得人高马大，虽然瘦，还是可以伪装得很正派。阿鹤那个时候没那么高，1000米长跑比我还不如，不过挺乐观，就是不停得跑而已，喘得厉害就走两步，其实这个很少人看到。喜欢听歌，偶像周建华（这里我是故意的，应该是周华健），唱歌的时候身姿非常曼妙，臀部不时跳动，恐怕这个，他自己都不知道吧。

食物

CJ 的家大家都很喜欢去，周末、放假经常去。两三个，一两堆，坐满了凳子，站在窗边，看街边美女，还有非常没有礼貌的坐床上的，只是那时没什么介怀，也不懂，都是小毛孩儿。记得有一次，中考还是终考，已经忘了，中午吃饭自己解决，某同学提议，买泡面去CJ家吃。说实话，泡面这东西，现在看来，早已不再新鲜，有时候还会认为吃多了对身体不好。

可那时候，不一样，记得买的时统一的鲜虾鱼板面，边煮边咽口水，两三个人居然还议论蔬菜包里有什么，黑黑的锅子里面，随着开水的翻滚，弯曲的面条是不是翻滚上来，当时的感觉就是人间美味，围着CJ家的小方桌，可以把汤水都喝完，吃完了还特地表现得非常老练的把碗洗掉。在这之后吃的鲜虾鱼板面，总没有这味道。

阿鹤的家，不是经常去，有段暑假经常去，不知道哪里搞来霸王的大陆，确实是非常经典的游戏，在阿鹤家吃中饭，吃了一道非常美味的菜，以后吃的都没这个味道，甚至做法也没有，特别奇怪，经典菜肴，番茄炒蛋，据说是他妈妈烧的，我看他也不像烧菜的主，番茄比较大块，蛋不老，也不大，感觉应该番茄翻炒的时间比较久，就有特别的味道。后来吃的，基本没那感觉，好像番茄催熟的多了，而且其他做法，番茄都被炒很烂，根本夹不起。

玩物

和阿鹤一起玩的霸王大陆，确实是一款好作品，非常起兴，关键，那卡好像记忆有问题，阿鹤很拼，直接就不关机，两个人莫名其妙玩了好多天，实在很佩服自己，后来一次失误，记忆全没了。不过阿鹤兄实在很拼，不知道哪里搞来了的秘籍，玩长的不行，我们玩短的，貌似这样更爽点，也不知道这什么心态，反正是玩得爽爽的。

CJ家的就相对高级点，当时风靡的游戏，Diablo 2， 酷到不行，一起买了7K多的主机，记得创新的音响，本博现在还在用呢，比较帅气，CJ是外貌协会的，Diablo 2 拿回来 CG动画先观摩下，之后就展开厮杀，本博做在旁边帮忙加血，看着画面也很激动。这感觉，估计再也找不回来了。

事物

CJ 和 阿鹤 是活宝，加上本博，在网络游戏兴起时，暑假相约去网吧，相约一起车站碰头，现在感觉坐蛮久的公交车，还要走上一段路，才来到网吧， 相当之矜持，CJ是最认真的一个，研究各种攻略，打怪升级也不嫌麻烦，网络游戏，开始是非常有趣的，新人村里出来，衣服都没有，开始打怪。阿鹤可能跟本博一样，比较怕麻烦，因此，最开心的事是合伙打人，也不在乎等级，就凭两个人，去打一个，现在的话是开黑，记忆非常深刻的是，一个网名叫 “火狐火狐” 的，貌似连杀好几次，后来看到我们就跑。

除了玩游戏，最多做的是夏天的晚间散步，迎着微风，憧憬着未来。譬如桥上，风吹着面孔，就比较容易憧憬，幻想；譬如河边，新建的健身设施，就比较容易聊些不入流的猥琐小段子，笑得如此邪恶；譬如小区里，最适合偶遇，可惜去了好多次，也没有偶遇过一次。

说着心中的秘密，聊着人家的长短，那一年的岁月如此的羞涩，如此美丽。那一年的夕阳红得如此的鲜艳，路边的蚊子也没少吃。那一年的空气，流淌着青春的躁动，风中夹杂着自然的气息。那一年，我们美丽的青春，淡淡的而又不同凡响的青春。

致青春与梦想，给未来的自己，未来的朋友们。

博客再一次搬家 目的地 Linode

前端时间比较忙碌，手头几个比较难办的难点需要攻克，这是最近少有更新博客的其中一个原因。

域名被封

另外一个也是比较重要的原因，就是本博的阿里云VPS被要求备案，也就是域名 www.jacky-wu.com ，本来还有点侥幸心理，像本博这种小站点，个人用的，应该不会惊动某某某吧。

购买新域名，新空间

因此，在被封了域名之后，决定再买个域名，听说只要域名在国外注册，就不需要备案，于是乎，立马买了 www.jackylab.com 。这个比较好，因为在移动设备上，输入 连字符 - 还是挺麻烦的，如果全部是字母，那体验还是不错的。

新域名是在 godaddy上买的，绑定之后，一切正常，大概有大半天时间都是可以访问的，可是到了第二天，本博就傻眼了，还是被封了。

话说本博是个很懒的人，懒得去备案，另外也是白羊座，比较冲动，立马 linode 买起，虽说冲动，为了表现本博成熟的一面，在买之前，搜了一把优惠码， 嗯，10 刀， 还是挺好的，1个月的优惠啊。

因为之前有涉外工作的关系，有张双币信用卡，立马拿到 VPS， Linode 最小的那款，不过个人用已经很不错了， 1G的内存， SSD 24GB， 都很 OK， 以前折腾 CentOS， 这次为了省事，Ubuntu Server 了。 机房在东京，速度还不错

未来继续折腾

换 Server 主要是将精力放到业务上，目前的博客系统比较粗糙， Flask 做的，算是玩一把，目前源码放在 github，偶尔有所更新。

折腾了这么多，其实给本博另外的思考，决定暂时放弃脚本语言，专心到 Java 和 C/C++ 上来，Java是因为 Android， 目前工作内容就是 Android开发， 另外，所有的模式以及UML，本博目前都是以Java为蓝本。

另外一个重要原因是某小盆友从 PHP转到 Java去了，弄得本博心动，顺手搭了 Sping MVC + Mybatis + Tiles 的小应用，并且做了个 Android的客户端，因为这个被人看上了，无法开源，那么，本博决定，接下来的工作，就是用以上技术，再折腾一次这个博客，俗话说的好，人生在于折腾，这次折腾必定开源出来，尽请期待吧。。。
