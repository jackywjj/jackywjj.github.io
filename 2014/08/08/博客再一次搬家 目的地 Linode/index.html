博客再一次搬家 目的地 Linode

前端时间比较忙碌，手头几个比较难办的难点需要攻克，这是最近少有更新博客的其中一个原因。

域名被封

另外一个也是比较重要的原因，就是本博的阿里云VPS被要求备案，也就是域名 www.jacky-wu.com ，本来还有点侥幸心理，像本博这种小站点，个人用的，应该不会惊动某某某吧。

购买新域名，新空间

因此，在被封了域名之后，决定再买个域名，听说只要域名在国外注册，就不需要备案，于是乎，立马买了 www.jackylab.com 。这个比较好，因为在移动设备上，输入 连字符 - 还是挺麻烦的，如果全部是字母，那体验还是不错的。

新域名是在 godaddy上买的，绑定之后，一切正常，大概有大半天时间都是可以访问的，可是到了第二天，本博就傻眼了，还是被封了。

话说本博是个很懒的人，懒得去备案，另外也是白羊座，比较冲动，立马 linode 买起，虽说冲动，为了表现本博成熟的一面，在买之前，搜了一把优惠码， 嗯，10 刀， 还是挺好的，1个月的优惠啊。

因为之前有涉外工作的关系，有张双币信用卡，立马拿到 VPS， Linode 最小的那款，不过个人用已经很不错了， 1G的内存， SSD 24GB， 都很 OK， 以前折腾 CentOS， 这次为了省事，Ubuntu Server 了。 机房在东京，速度还不错

未来继续折腾

换 Server 主要是将精力放到业务上，目前的博客系统比较粗糙， Flask 做的，算是玩一把，目前源码放在 github，偶尔有所更新。

折腾了这么多，其实给本博另外的思考，决定暂时放弃脚本语言，专心到 Java 和 C/C++ 上来，Java是因为 Android， 目前工作内容就是 Android开发， 另外，所有的模式以及UML，本博目前都是以Java为蓝本。

另外一个重要原因是某小盆友从 PHP转到 Java去了，弄得本博心动，顺手搭了 Sping MVC + Mybatis + Tiles 的小应用，并且做了个 Android的客户端，因为这个被人看上了，无法开源，那么，本博决定，接下来的工作，就是用以上技术，再折腾一次这个博客，俗话说的好，人生在于折腾，这次折腾必定开源出来，尽请期待吧。。。
